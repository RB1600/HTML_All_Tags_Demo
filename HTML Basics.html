Easy Explanation of Each Tag

  • <!DOCTYPE html>: Declares this file is an HTML document.
  • <html>: Wraps all your HTML code.
  • <lang="en">: Specifies the language of the HTML document (English in this case).
  • < and >: Angle brackets are used to enclose HTML tags. 'lt' means less-than and 'gt' means greater-than.
  • lang="en": Sets the language of the page to English.
  • <head>: Contains info about the page (not visible on the page).
  • <meta charset="UTF-8">: Sets the character encoding so all symbols show correctly.
